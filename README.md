# Integración de Base de Datos con WooCommerce

Este repositorio contiene recursos y guías para la integración y gestión de bases de datos en entornos de comercio electrónico utilizando WooCommerce.

## Contenido

### Documentación Técnica

- **[Guía: Creación e Integración de Base de Datos para WooCommerce](./WOOCOMMERCE_DB_TUTORIAL.md)**
  - Configuración de base de datos.
  - Conexión con WordPress.
  - Estructura de tablas de WooCommerce (`wp_wc_orders`, `wp_postmeta`).
  - Integración con tablas personalizadas.

## Requisitos del Sistema

- **WordPress**: Versión 6.0 o superior
- **WooCommerce**: Versión 8.0 o superior (HPOS recomendado)
- **Base de Datos**: MySQL 5.7+ o MariaDB 10.3+
