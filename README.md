# Integración de Base de Datos PostgreSQL con WooCommerce

Este repositorio contiene recursos y guías para integrar y gestionar bases de datos PostgreSQL en entornos de comercio electrónico utilizando WooCommerce.

## Contenido

### Documentación Técnica

- **[Guía: Integración de PostgreSQL con WooCommerce](./WOOCOMMERCE_DB_TUTORIAL.md)**
  - Requisitos y configuración del entorno PostgreSQL.
  - Instalación del controlador de base de datos (PG4WP).
  - Estructura de tablas para PostgreSQL.
  - Desarrollo de consultas compatibles y tablas personalizadas.

## Requisitos del Sistema

- **WordPress**: Versión 6.0 o superior.
- **WooCommerce**: Versión 8.0 o superior.
- **Base de Datos**: PostgreSQL 12 o superior.
- **Extensión PHP**: `pgsql` o `pdo_pgsql`.
